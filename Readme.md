**Обзор**

Проект Pink это вёрстка сайта сервиса загрузки фотографий и где можно скачать приложение.   
Сайт адаптирован для desktop, pad и mobile

**Технологии**

* __HTML5__
* __Sass__
* __CSS3__
* __Flexbox__
* __Семантическая__ вёрстка
* __Gulp__
